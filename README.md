# Научиться учиться :mortar_board:

  Данный проект является результатом изучения вёрстки на HTML и CSS с использованием методологии **БЭМ**. Представляет собой статичный сайт.

  ## Используемые технологии

  * HTML5
  * iframe
  * CSS3
  * CSS animations
  * CSS Flexbox
